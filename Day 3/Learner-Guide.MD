## üèÅ 1. Object-Oriented Programming (OOP)

### What is OOP?
- A programming paradigm using **objects** and **classes** to model real-world entities.  
- Encourages modular, reusable code.

### Class and Object Example
```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print(f"Hello, my name is {self.name} and I am {self.age} years old.")

# Create objects
person1 = Person("Alice", 25)
person2 = Person("Bob", 30)

person1.greet()
person2.greet()
````

## ‚ûï 2. Python Libraries

### Standard Libraries

* **random**: generate random numbers, shuffle lists
* **datetime**: work with dates and times
* **math**: mathematical functions

```python
import random
print(random.randint(1, 10))

from datetime import datetime
print(datetime.now())

import math
print(math.sqrt(16))
```

### Optional Libraries

* **requests**: access web APIs
* **pandas**: handle data in tables (DataFrames)

```python
import pandas as pd

data = {"Name": ["Alice", "Bob"], "Age": [25, 30]}
df = pd.DataFrame(data)
print(df)
```

### Exercises

2. Use `datetime` to print current date and time in `YYYY-MM-DD HH:MM` format.
