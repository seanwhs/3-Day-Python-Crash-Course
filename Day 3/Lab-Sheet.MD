### Task 1: Create a `Car` Class
1. Define a `Car` class with attributes: `make`, `model`, `year`.

### Task 3: Optional Challenge

- Add methods: `deposit(amount)` and `withdraw(amount)`.  
- Test by creating accounts and performing transactions.
