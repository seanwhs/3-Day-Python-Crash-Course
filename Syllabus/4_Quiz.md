# Quiz - Control Structures

1. What is the syntax for an if-else statement in Python?  
a. if condition:  
else:  
b. if condition then:  
else:  
c. if condition:  
elif:  
else:  
d. if condition:  
else if:  
else:  

Answer: a. if condition:  
else:  

1. Which loop is used to iterate over a sequence of elements for a specified number of times?  
a. if loop  
b. while loop  
c. for loop  
d. do-while loop  

Answer: c. for loop  
  
1. What is the output of the following code?  

```
pythonCopy code
num = 7
while num > 0:
    print(num)
    num -= 2

```

a. 7 5 3 1  
b. 1 3 5 7  
c. 7 4 1  
d. 7 5 3  

Answer: a. 7 5 3 1  

1. Which of the following is a conditional expression in Python?  
a. if-else statement  
b. for loop  
c. while loop  
d. ternary operator  

Answer: d. ternary operator
