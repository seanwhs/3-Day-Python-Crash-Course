# Quiz - OOP

1. What is object-oriented programming?  
a. A programming paradigm that focuses on functions.  
b. A programming paradigm that focuses on objects.  
c. A programming paradigm that focuses on procedures.  
d. A programming paradigm that focuses on data types.  

Answer: b. A programming paradigm that focuses on objects.  

1. What is an object in object-oriented programming?  
a. A function that performs a specific task.  
b. A data type that stores values.  
c. A collection of related data and methods.  
d. A control structure that makes decisions based on conditions.  

Answer: c. A collection of related data and methods.  

1. What is a class in object-oriented programming?  
a. An instance of an object.  
b. A type of data that can store values.  
c. A template for creating objects.  
d. A method that is shared by all objects.  
  
Answer: c. A template for creating objects.  

1. What is inheritance in object-oriented programming?  
a. The ability to define a new class based on an existing class.  
b. The ability to create a new object based on an existing object.  
c. The ability to define a new method in an existing class.  
d. The ability to define a new data type in an existing class.  

Answer: a. The ability to define a new class based on an existing class.  

1. What is polymorphism in object-oriented programming?  
a. The ability to define multiple methods with the same name in a class.  
b. The ability to create multiple objects with the same data and methods.  
c. The ability to define a new class based on an existing class.  
d. The ability to define a new method in an existing class.  

Answer: a. The ability to define multiple methods with the same name in a class.  

1. What is the purpose of encapsulation in object-oriented programming?  
A) To hide implementation details  
B) To provide a public interface for a class  
C) To allow inheritance between classes  
D) To manage memory allocation  

Answer: A  

1. What is inheritance in object-oriented programming?  
A) When a class is defined within another class  
B) When a class is copied to another class  
C) When a subclass inherits the attributes and methods of a superclass  
D) When a superclass inherits the attributes and methods of a subclass  

Answer: C  
  
1. What is polymorphism in object-oriented programming?  
A) The ability of a subclass to override the methods of a superclass  
B) The ability of an object to take on many forms  
C) The ability of a class to implement multiple interfaces  
D) The ability of a class to create objects of another class  

Answer: B  

1. What is a constructor in object-oriented programming?  
A) A method used to create a new object of a class  
B) A method used to modify the behavior of an object  
C) A method used to delete an object  
D) A method used to encapsulate data in a class  

Answer: A  

1. What is a method in object-oriented programming?  
A) A function that operates on the attributes of an object  
B) A constructor used to create a new object of a class  
C) A variable that is shared among all instances of a class  
D) An attribute that describes the state of an object  

Answer: A
