# Quiz - Input and Output

1. How do you write to a file in Python?  
A. Using the write() function  
B. Using the read() function  
C. Using the open() function  
D. Using the input() function  

Answer: A. Using the write() function  

1. What method do you use to read the entire contents of a file in Python?  
A. read()  
B. readline()  
C. readlines()  
D. for loop  

Answer: A. read()  

1. Which of the following is true about binary mode in file handling?  
A. It is used to handle text files  
B. It is used to handle binary files  
C. It is used to handle both text and binary files  
D. It is not a valid mode in file handling  

Answer: B. It is used to handle binary files  

1. What is the function used to check if a file exists in Python?  
A. exists()  
B. open()  
C. read()  
D. write()  

Answer: A. exists()  

1. What is the default mode for opening a file in Python?  
A. read-only mode (r)  
B. write-only mode (w)  
C. read-write mode (r+)  
D. binary mode (b)  

Answer: A. read-only mode (r)  

1. What is the function used to open a file in Python?  
A. open()  
B. read()  
C. write()  
D. close()  

Answer: A. open()  

1. What is the mode used for opening a file for writing in Python?  
A. r  
B. w  
C. a  
D. x  

Answer: B. w  

1. How do you read a file line by line in Python?  
A. Using the read() function  
B. Using the readline() function  
C. Using the readlines() function  
D. Using a for loop  

Answer: D. Using a for loop  

1. How do you close a file in Python?  
A. close()  
B. exit()  
C. end()  
D. quit()  

Answer: A. close()  

1. What mode should you use to open a file for reading and writing in Python?  
A. r  
B. w  
C. a  
D. r+  

Answer: D. r+  

1. Which of the following is a valid mode to open a file in Python?    
A. "read"  
B. "write"  
C. "append"  
D. All of the above  

Answer: D. All of the above  

1. What is the difference between read() and readline() methods for file objects in Python?  
A. read() reads the entire file, while readline() reads one line at a time  
B. readline() reads the entire file, while read() reads one line at a time  
C. read() and readline() are used for the same purpose  
D. None of the above  

Answer: A. read() reads the entire file, while readline() reads one line at a time  

1. What is the syntax to write to a file in Python?  
A. file.write("text")  
B. write("text")  
C. file.read("text")  
D. read("text")  

Answer: A. file.write("text")  

1. Which method is used to close an opened file in Python?  
A. close()  
B. end()  
C. exit()  
D. quit()  

Answer: A. close()  

1. Which of the following is an example of file mode for reading and writing in Python?  
A. "w+"  
B. "a+"  
C. "r+"  
D. "x+"  

Answer: C. "r+"
