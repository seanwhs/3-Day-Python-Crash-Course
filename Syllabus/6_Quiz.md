# Quiz - Modules

1. What is a module in Python?  
A. A collection of related modules and sub-packages.  
B. A single file containing Python definitions and statements.  
C. A built-in function or method in Python.  
D. A variable that stores a value or reference.  

Answer: B  

1. What is a package in Python?  
A. A built-in function or method in Python.  
B. A single file containing Python definitions and statements.  
C. A collection of related modules and sub-packages.  
D. A variable that stores a value or reference.  

Answer: C  

1. Which file is required to make a directory a Python package?  
A. **init**.py  
B. main.py  
C. package.py  
D. **main**.py  

Answer: A  

1. How do you import a module named "my_module" in Python?  
A. **`import my_module`**  
B. **`from my_module import *`**  
C. **`import my_module as mm`**  
D. All of the above  
  
Answer: D  

1. Which statement is used to import a specific function from a module?  
A. **`import module`**  
B. **`from module import *`**  
C. **`from module import function`**  
D. **`import function from module`**  

Answer: C  

1. How do you check the list of built-in modules in Python?  
A. Use the **`dir()`** function  
B. Use the **`help()`** function  
C. Use the **`modules`** attribute of the **`sys`** module  
D. Use the **`builtins`** module  

Answer: C  

1. Which keyword is used to define a module in Python?  
A. **`package`**  
B. **`module`**  
C. **`def`**  
D. **`class`**  

Answer: B  

1. How do you reload a module in Python?  
A. **`import module`**  
B. **`reload(module)`**  
C. **`importlib.reload(module)`**  
D. **`importlib.import_module(module)`**  

Answer: C  

1. What is the purpose of the **`if __name__ == "__main__":`** statement in a Python module?  
A. To run the module as a standalone program  
B. To import the module into another program  
C. To define a function in the module  
D. To declare a variable in the module  

Answer: A  

1. Which statement is used to access a function in a module?  
A. **`module.function()`**  
B. **`function(module)`**  
C. **`import module.function`**  
D. **`function.import(module)`**  

Answer: A  

1. What is the difference between a built-in module and a third-party module in Python?  
A. Built-in modules are created by third-party developers, while third-party modules are created by the Python development team.
B. Built-in modules are included in the Python standard library, while third-party modules are not.  
C. Built-in modules are installed using **`pip`**, while third-party modules are pre-installed in Python.  
D. There is no difference between a built-in module and a third-party module.    

Answer: B  

1. What is the purpose of a virtual environment in Python?  
A. To create a separate environment for each module  
B. To isolate Python packages and dependencies for each project  
C. To install multiple versions of Python on the same machine  
D. To run Python code on a virtual machine  

Answer: B  

1. Which command is used to create a virtual environment in Python?  
A. **`pip install virtualenv`**  
B. **`python -m venv myenv`**  
C. **`virtualenv myenv`**  
D. **`venv create myenv`**  

Answer: B  

1. Which command is used to activate a virtual environment in Python?  
A. **`source myenv/bin/activate`**  
B. **`python activate myenv`**  
C. **`venv activate myenv`**  
D. **`myenv/activate`**  

Answer: A  

1. Which command is used to install a package in a virtual environment?  
A. **`pip install mypackage`**  
B. **`python install mypackage`**  
C. **`virtualenv install mypackage`**  
D. **`myenv/pip install mypackage`**  

Answer: A
