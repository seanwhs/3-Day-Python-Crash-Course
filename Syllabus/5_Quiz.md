# Quiz - Functions

1. What is a function in Python?  
a. A keyword used to define a block of code  
b. A block of code that performs a specific task  
c. A way to store data in memory  
d. A type of variable that holds a sequence of characters  

Answer: b  

1. How do you define a function in Python?  
a. Using the def keyword followed by the function name and parameters in parentheses  
b. Using the function keyword followed by the function name and parameters in parentheses  
c. Using the define keyword followed by the function name and parameters in parentheses  
d. Using the function keyword followed by the function name and parameters in braces  

Answer: a  

1. How do you call a function in Python?  
a. By typing the function name followed by the arguments in braces  
b. By typing the function name followed by the arguments in parentheses  
c. By typing the function name followed by the arguments in brackets  
d. By typing the function name followed by the arguments in angle brackets  

Answer: a  

1. What is the difference between a parameter and an argument in Python?  
a. There is no difference, they mean the same thing  
b. A parameter is a value passed into a function, while an argument is a variable used to define a function  
c. A parameter is a variable used to define a function, while an argument is a value passed into a function  
d. A parameter and an argument are both types of variables  
  
Answer: c  

1. What is a lambda function in Python?  
a. A function that takes a variable number of arguments  
b. A function that returns a Boolean value  
c. A function that can be defined in one line of code  
d. A function that is called recursively  

Answer: c  

1. What is the purpose of the return statement in a function?  
a. To stop the execution of the function  
b. To return a value from the function to the calling code  
c. To skip over certain parts of the function  
d. To print a message to the console  

Answer: b  

1. What is the scope of a variable in Python?  
a. The region of the program where the variable can be accessed  
b. The value of the variable  
c. The data type of the variable  
d. The memory location of the variable  

Answer: a

a. Global variables can be accessed anywhere in the program, while local variables are only accessible within the function in which they are defined  
b. Local variables can be accessed anywhere in the program, while global variables are only accessible within the function in which they are defined  
c. Global variables have a smaller scope than local variables    
d. Local variables have a smaller scope than global variables

1. What is recursion in Python?    
a. A function that calls itself    
b. A type of loop  
c. A data type  
d. A way to handle errors in code    

Answer: a  

1. What is the purpose of the map() function in Python?  
a. To apply a function to each element of an iterable
