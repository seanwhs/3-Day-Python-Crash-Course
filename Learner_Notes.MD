## 1ï¸âƒ£ Variables & Data Types

### What is a Variable?

A variable is a **label pointing to a value in memory**.

Python is **dynamically typed**:

* You donâ€™t declare types
* Python infers them at runtime

```python
x = 10        # int
x = "ten"     # str (same variable, new type)
```

### Core Built-in Types

| Type  | Example | Use Case             |
| ----- | ------- | -------------------- |
| int   | 42      | counters, IDs        |
| float | 3.14    | prices, measurements |
| str   | "hello" | text, messages       |
| bool  | True    | flags, conditions    |

```python
price = 19.99
is_logged_in = False
```

### Mutable vs Immutable (VERY IMPORTANT)

* **Immutable:** int, float, str, tuple
* **Mutable:** list, dict, set

```python
x = 5
y = x
y += 1
# x is still 5
```

```python
lst1 = [1, 2]
lst2 = lst1
lst2.append(3)
# lst1 is now [1, 2, 3]
```

ğŸ§  **Bootcamp Insight:**
Most beginner bugs come from misunderstanding mutability.

## 9ï¸âƒ£ Decorators (Power Tool)

Decorators **wrap functions** to add behavior.

```python
def logger(func):
    def wrapper():
        print("Before")
        func()
        print("After")
    return wrapper
```

```python

def run():
    print("Running")
```

### Real Uses

* Authentication
* Logging
* Timing
* Caching
