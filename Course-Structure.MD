# **Weekend Python Crash Course – Content Package**

## **Day 1: Python Basics & Core Concepts**

| Content Type             | File Name Suggestion    | Purpose                                                                                                         |
| ------------------------ | ----------------------- | --------------------------------------------------------------------------------------------------------------- |
| Learner Guide / Workbook | `Day1_LearnerGuide.pdf` | Step-by-step explanations, examples, exercises for Python intro, IDE setup, variables, data types, control flow |
| Lab Sheet / Exercises    | `Day1_LabSheet.docx`    | Hands-on exercises: running scripts, input/output, if/else statements, loops                                    |
| Slide Deck               | `Day1_Slides.pptx`      | Visual explanations, diagrams for data types, loops, IDE screenshots                                            |
| Solution Key             | `Day1_Solutions.pdf`    | Completed exercises with explanations                                                                           |
| Cheat Sheet (Optional)   | `Day1_CheatSheet.pdf`   | Quick reference for Python syntax, operators, basic functions                                                   |
| Demo Scripts             | `Day1_DemoScripts.py`   | Ready-to-run code for live demonstration (Hello World, loops, conditional statements)                           |
